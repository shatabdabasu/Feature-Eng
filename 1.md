## 1. Feature Transformation
Modifying existing features to make them more suitable for machine learning models.

### Missing Value Imputation
- Remove missing values
- Mean imputation
- Median imputation
- Mode imputation

### Handling Categorical Values
- Convert categorical data into numerical form
- Example: One-Hot Encoding

### Outlier Detection
- Identify and remove data points that significantly differ from others
- Important for algorithms sensitive to extreme values

### Feature Scaling
- Normalize feature ranges
- Important for distance-based algorithms (e.g., KNN)

---

## 2. Feature Construction
Creating new features from existing ones using domain knowledge.

Example:
- `family_size = siblings/spouses + parents/children`

---

## 3. Feature Selection
Selecting the most relevant features for model building.

Goals:
- Reduce dimensionality
- Improve model performance
- Reduce training time
- Remove less important features

---

## 4. Feature Extraction
Creating new lower-dimensional features from high-dimensional data using algorithms.

Example:
- **PCA (Principal Component Analysis)**

Use case:
- High-dimensional datasets such as images
