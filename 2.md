# Feature Scaling – Quick Notes

## What is Feature Scaling
Technique used to bring independent features to a **similar numerical range**.

Example  
- IQ → ~100s  
- CGPA → ~10s  
Scaling makes them comparable.

---

## Why Feature Scaling is Needed
- Some algorithms use **distance calculations**.
- Features with larger values can **dominate the model**.

Example  
- KNN distance calculations  
- Age vs Salary problem

Goal:
- Prevent bias toward large-value features.

---

## Types of Feature Scaling
1. **Standardization (Z-score normalization)**
2. **Normalization**
   - Min-Max Scaling
   - RobustScaler

---

# Standardization

## Formula
\[
X' = \frac{X - \mu}{\sigma}
\]

Where:
- μ = mean  
- σ = standard deviation  

Result:
- **Mean = 0**
- **Standard deviation = 1**

---

## Intuition

### Mean Centering
Shift data so feature mean becomes **0**.

### Scaling by Standard Deviation
Compress or expand data so **std = 1**.

Important:
- **Distribution shape does NOT change**
- Only **center and scale change**
